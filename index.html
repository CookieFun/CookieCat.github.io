DESC深度自适应聚类

发表于 2019-11-15

深度自适应聚类

Intro

传统的聚类方法通常聚焦于两个问题：

  1. 如何选择正确的距离度量去衡量样本之间的相似度
  2. 如何定义正确的聚类模式去描述数据的特征

对于第一部分，距离度量通常取决于数据的特征空间，这一部分的方法通常是无监督的。早期的方法通常是定义特征描述算子，比如针对图像的HOG、SIFT算子。然而这些传统方法通常没有很强的适应性，难以区分物体和背景。为了克服这一问题，近期许多深度无监督学习的方法被提出，比如自编码器，去噪自编码器等。这些方法通过对数据中潜在的模式进行自编码，用重构的方式训练整个模型。尽管这种方法已经取得了很好的效果，可是在表示高位特征的数据时还有很大的提升也能够空间。

对于第二部分，传统方法通过不同的可理解的模式对数据进行描述。基于这个思想，高斯混合分布是一个常用的描述方法。同时，K-means、K-means++、DBSCAN也是常用的聚类模式。这些模式在不同的数据集上都取得过很好的效果，但是没有一种模式可以做到对不同数据的适应性。

于是，大部分聚类方法都是基于这两个问题进行设计的，其聚类过程也通常是两阶段的。这就造成了一个问题，那就是在衡量相似度的过程不能很好第适应聚类过程，而聚类的时候也不能继续改进相似度的衡量。

为了同时进行对数据的高维特征学习和聚类运算，本文提出了一种深度自适应聚类方法。这种方法通过把聚类问题转换成两两样本之间的二分类问题，首先，模型会将每个样本编码成一个k维的概率分布向量(和为1)，通过自适应的训练过程，让模型逐渐学会区分两个样本是否属于同一个聚类，最终，每个样本对应的分布向量就是它的类标信息。

Model

第一部分，深度特征提取。将每个样本输入到共享参数的一个深度神经网络模型中，通过卷积、池化等操作，最终将样本输出为一个k维的特征向量，这个向量的意义是这个样本属于每一个聚类的概率。在训练开始阶段，这个特征向量的各维的值都较为平均，随着训练的进行，这个特征向量会逐渐逼近一个one-hot向量，即我们最终要求的类别信息。为了便于计算，通常我们要给I向量一个约束，本文中用的是P范式约束。

$$数据集：D={(x_i,x_j,r_{ij})}_{i=1,j=1}^n$$

$$DNN模块：f(x_i;w)=I_i$$

$$I向量满足约束C_p：||I||_p=1$$

第二部分，将样本两两输入到共享参数的DNN模型中，得到两个分布向量，通过内积的方法确定两个向量的相似度，根据相似度的大小和阈值的关系，确定这对样本的标签rij（1表示在同一个聚类，0表示不在同一个聚类）

$$比较模块：g(x_i,x_j;w)=S(I_i,I_j)=I_i\cdot I_j=f(x_i;w)\cdot f(x_j;w)$$

$$r_{ij}的确定：r_{ij}=\left{\begin{matrix} 1, & S(I_i,I_j)>u\ 0, & S(I_i,I_j)<l\ None, & otherwise \end{matrix}\right.$$

第三部分，将样本两两输入模型得到的相似度和标签做比较，计算误差，通过反向传播修改DNN的参数。

$$指标系数：v_{ij}=\begin{cases} 1, & r_{ij}=0,1 \ 0 ,& r_{ij}=None\end{cases}$$

$$损失函数：E(w)=\sum_{i=1}^n\sum_{j=1}^nv_{ij}L(r_{ij},g(x_i,x_j;w))=\sum_{i=1}^n\sum_{j=1}^nv_{ij}|r_{ij}-g(x_i,x_j;w)|^2$$

Experiment

可以看到在大部分数据集上，该方法相比传统的方法有着明显的提高，并且该方法在不同类型的数据集（图像、语音、文本）上都取得了较高的准确率。

论文中展示了随着训练轮数的增加，不同类别的样本的特征向量的分布情况。可以看到当训练结束后，在特征空间中很好地将样本进行了划分。

Conclusion

这篇论文提出了一个很好的基于深度学习的聚类模式，将聚类问题转化成特征提取和样本对的二分类问题的结合，很好地将深度学习的优势融入到了聚类当中。然而，该文章中的模型尚存在以下几个问题：

  1. 参数的初始化以及初始几次计算样本的选取对模型的参数影响很大，如果这部分的初始化没做好，整个模型的效果将大打折扣。所以需要一种稳定性更强的计算方法。
  2. 由于需要对样本进行两两比较，那么遍历全量的样本对空间所需的时间复杂度将大大提升，如何减小计算复杂度是另一个需要考虑的问题。

Round G 2019 - Kick Start 2019

发表于 2019-11-15

有一段时间没做题了，手生了。

Book Reading

有$n$本书，标号$1$到$n$，其中有$m$页被撕掉，分别是$p_1,p_2,…,p_m$。现在有$q$个人读这本书，第$j$个人读的页数是$r_j$的倍数。问这些人总共读了多少页（同一页可以被读多次）。$n,m,q\leq 1e5$

假设没有缺失的页，那么每个人读的页数是$n/r_j$。现在只要计算每个缺页会被多少个人读到。如果直接枚举每个$r_j$，再去找$r_j$的倍数的话，复杂度是$O(q\times n)$。但是我们发现$n/1+n/2+n/3+…+n/n = nlogn$，于是我们枚举$1..n$每一页，将读这一页倍数的人数向后累加，最后统计答案即可。

The Equation

求使得$(A_1 \oplus k) + (A_2 \oplus k) + … (A_N \oplus k) ≤ M, $最大的$k$。$A_i, M \leq 1e15$

既然是求最大的k，那么我们从高位到低位考虑，肯定是高位尽量选1。假设所有$A_1…A_N$中第k位为0的数的个数为$cnt$，如果第k位选了1，那么总和要增加$(n-cnt) \times 2^{k-1}$，否则增加$cnt \times 2^{k-1}$。所以我们只需要维护前k-1位最小的前缀和，用于判断第k位是否可以选1即可。

Shift

有N个时间段，两个人，每个人在第i个时间段工作的幸福度分别为$A_i$和$B_i$。求在每个时间段至少有一个人工作的情况下，两个人的幸福度和都不小于h的方案数。$N\leq 20, h\leq 1e9$

由于枚举所有方案需要$3^{20}$次计算，肯定是不行的。所以我们考虑先枚举A的工作方案，然后再看B有多少可行的工作方案与之对应。
我们设1为工作，0为不工作，那么假设B的一个可行方案是$\hat s=00101$，那么它可以对应的最小的A的方案$s$应该是$11010$，并且它可以对应任何$11010$的超集，比如$11110$,$11011$等。我们设$f[s]$表示A的方案为$s$时对应多少可行的B的方案。那么对于每个$s$来说，我们先检查它对应的最小的B方案$\hat s$是否可行，如果可行，就用$s$去更新所有$s$的超集。这里为了避免重复更新，我们可以限制更新路径为从低位到高位增加，也就是先枚举位，在枚举这将这一位变成1的转移。

官方题解给的答案是将原序列分成两半，计算两边分别能组成的幸福度序列$<S_a, S_b>$，$ <T_a, T_b>$然后求答案时，等价于对每个$ <T_a, T_b>$，查找有多少个$<S_a, S_b>$满足$h-S_a\leq T_a$ 且 $h-S_b\leq T_b$，等价于在一个二维平面上查找一个点的左下角的点的个数。二维偏序问题，排序+树状数组即可。

论文内容梳理

《光学遥感卫星影像云检测方法及应用》（武汉大学）

论文：http://cdmd.cnki.com.cn/Article/CDMD-10486-1018196776.htm

1.云检测基础知识及现状

  1. 光学遥感影像的分类（超高分辨率、高分辨率、中分辨率、低分辨率）
  2. 云的成因及分类
    根据云底部的高度：高云、中云、低云
    根据云的基本形态：积层云（局部的、孤立的、有云影），层状云（连片存在、面积大、厚薄均匀、边缘可见云影），波状云（厚薄不一、连片存在、有云缝、无云影）
    根据卫星影像上的外观：厚云（完全遮挡地表场景，包括浓雾和航迹云），薄云（透光，造成影像高度升高，但基本不影响判读）
  3. 光学遥感卫星影像上云的特征
    辐射特征：
      1.更高的亮度值和更低的色度值
      2.积雪冰川和部分城市建筑物有和云类似的亮度和色度值
      3.同类型的地表地物亮度分布比较集中，云的亮度分布比较分散
    几何特征：
    　　1.高分辨率影像上，云区内部纹理单一，边缘变化平缓，边界不清晰
    　　2.中分辨率影像上，波状云有着与地表物体类似的纹理，积状云有与地表物体类似的边缘
    　　3.云没有固定形态
    时相特征：
    　　云随时间变化最快，变化空间最大
    高程特征：
    　　相对高度是区分云和地表物体的最可靠特征，但仅通过单张光学遥感卫星图像无法获取各像素点的高程信息
  4. 国外遥感影像云检测算法
    　　1.亮度分析：基于云和地物亮度值的差异，通过阈值实现云区的提取。阈值可通过最大类间方差分类准则(Otsu阈值)获得
    　　2.色度分析：仅使用亮度会误判积雪沙石，需要辅以短波红外和热红外信息，MODIS可以做到
    　　3.纹理分析：分析云和地物纹理特征的差异，提取合适的特征，如分形维数，灰度共生矩阵，或者将特征在多尺度空间内进行延拓
    　　4.时相分析：基于同一地区时相相近的影像进行云检测，把云看作影像中变化的目标，常与基于单幅影像的云检测方法联合使用
    　　5.高程分析：通过多幅影像密集匹配技术得到云的三维几何特征，利用高程信息做区分，但执行成本和风险很高
    　　6.综合分类：K-means, 马尔可夫随机场, 卷积神经网络, SVM(结构风险最小化原理), 将多层语义分割和面向对象的思想引入上述分类模型，在顾及影像上下文信息的情况下，对粗分类结果进行多重判断以获得更高精度的云检测结果。
  5. 含云光学遥感卫星影像处理对策
    薄云增强：
    　　薄云造成影像亮度升高，清晰度下降，但没有对地物形成完全遮蔽，可通过影像增强方式进行修复，本质上时退化影像复原问题。常用方法有：直方图增强(拉伸、均衡化)、同态滤波、小波分析(云雾是低频部分，纹理处为高频部分)
    　　厚云场景的修复：多幅影像信息互补(像元替换、直方图匹配，线性回归)，影像镶嵌
    　　顾及云层感染的遥感影像产品制作：云掩膜

2.基于高斯混合模型的单幅卫星影像云检测方法，实现影像中云层范围的快速检测和自动识别

  1. 概要：
    研究单幅光学遥感影像快速自动云检测方法。使用高斯混合模型对影像直方图进行自动拟合，通过分析高斯混合模型中各分量的参数特征和临近关系，自适应地计算云与晴空之间地亮度阈值。然后通过形态学运算分析，消除阈值分割结果中的噪声，并优化调整云的轮廓，填充小面积云缝，使云区趋于联通的整体。
  2. 高斯混合模型的构建
    直方图预处理 -> 参数初始化 -> 参数拟合与约简 -> 分类数目调整
  3. 计算亮度阈值
    近似认为模型中的每一个高斯分量都表征一类地物，高斯分量的均值反映平均亮度值，标准差反映地物的亮度差异程度，权重反映这类地物在该副影像中所占有的像素数目。
  4. 计算“亮度-色度”阈值向量
    对于全色影像，由一维高斯混合模型求得亮度阈值，那么对于含有n个波段的多光谱影像，一个自然的想法是构建n维高斯混合模型，求得一个n维亮度阈值向量。但已知在可见光光谱范围内，云的反射率较高且趋近一致，在近红外光谱范围内，云的反射率开始降低，与晴空场景的区分度也随之降低。将n维模型简化为亮度和色度二维，其中亮度定义为可见光各波段（一般为红绿蓝三个波段）的最小亮度值，色度定义为可见光各波段最大和最小亮度值的差值。表征云的像元应该由较大的亮度值和较小的色度值，按照和单维混合模型的方式拟合。
  5. 云区优化与整合
    阈值分割缺点：
    　　会造成检测结果中大量高亮度、小面积噪声以及云层缝隙的存在
    基于超像素分割的优化整合：
    　　超像素可理解为由相似亮度、色度、纹理等特征的相邻像素构成的图像块。
    　　超像素分割即产生超像素的过程，使影像由像素级转变为区域级，在一定程度上有利于局部特征的提取与结构信息的表达，并降低处理对象的规模和后续处理的计算复杂度。
    　　超像素分割的相关算法：Mean-shift，SLIC
    　　优化整合策略：先得到基于像素级的分割结果，再通过超像素对其进行优化
  6. 基于形态学运算的优化整合
    提出5种形态学分析方法：腐蚀、膨胀、带限定条件的膨胀、面积统计、多边形分析
  7. 不足
    漏检：面积小且孤立存在的云；透光（可以观察到下垫面）的薄云
    误检：亮度大、面积大、形状不规则的人工建筑与自然地表

3.基于视觉显著性分析的云与高亮度人工目标区分方法

  1. 摘要：
    使用自适应二维Gabor滤波器提取高分辨率光学遥感影像上人工目标特有的直线边缘特征，作为视觉显著性的主要测度，在此基础上提取特征显著的区域，实现了高分辨率光学遥感影像上人工目标的自动识别，进而实现了云与高亮度人工目标的区分。
  2. 纹理与边缘
    在数字图像处理领域，“纹理”通常指图像局部不规则而宏观有规律的现象，“边缘”通常指图像局部亮度变化显著的区域，这些区域的灰度剖面存在跃变。特别地，城市区域有着丰富地纹理、更明显地边缘和更多的规则形状，植被和云层区域信息比较单一，没有明显的边缘和纹理。
  3. 视觉显著性常用测度特征
    a. 信息熵（平均信息量的表征）　　b. Canny算法（用于图像边缘检测）　　c. LSD算法（直线段检测分割算法）
  4. 面向人工目标识别的Gabor滤波器设计
    Gabor变换是一种窗口Fourier变换，当窗口Fourier变换的窗函数为高斯函数时，即为Gabor变换。Gabor变换对图像的边缘敏感，对光照变化不敏感，其数学形式是一个高斯函数和一个复变指数函数相乘。
    Gabor特征分为实部特征、虚部特征、复制特征。
    对于Gabor实部特征，较大的正值（对应亮条纹）和较小的负值（对应暗条纹）均代表较强的特征。
    Gabor特征能较好地对应遥感影像分辨率的变化，但需要针对性地调整滤波器参数。
    滤波器尺度参数的确定：当影像分辨率越低时，应选用较小的\(\lambda\)和\(\sigma\)，反之选用较大的\(\lambda\)和\(\sigma\)。
    滤波器窗口大小的确定：使用尽可能小的采样间隔和尽可能大的滤波窗口，具体策略在论文中给出。
  5. 基于Gabor特征显著性的人工目标识别
    1.特征图方向综合
    　即使用不同方向上的Gabor滤波器对影像执行多次卷积操作，然后综合结合各方向上Gabor特征的最大值。当核函数的方向与图像上直线纹理的方向一致或相近时，Gabor特征图上会变现处明显的明暗相间的条纹。
    2.特征图规则化
    　分析Gabor特征图的生成过程可推断，Gabor特征值的大小与影像边缘形态、滤波器参数和影像自身亮度有关。因此，将Gabor特征图做规则化处理，使之仅与影像中直线段边缘特征的强弱程度有关，而与滤波器参数和影像亮度均无关，且能增大区分度。
    3.特征图平滑滤波
    　使用高斯二维滤波器对特征图进行平滑，可将暗条纹与弱特征区域区分，并可在一定程度上解决大面积、亮度均匀人工目标中心特征较弱的问题。平滑步骤消除了特征图中的条纹现象，且膨胀扩充了特征显著区域，在一定程度上解决了面积较大、顶部亮度均匀建筑物Gabor特征较弱的问题。
    4.特征显著区域提取
    　规则化和平滑处理后的特征图与滤波器参数、影像亮度无关，可使用经验阈值对其进行分割，阈值以上的区域即为视觉显著区域。Otsu阈值以使类间方差最大、类内方差最小为准则，将图像分为目标和背景两种类型。论文设计经验阈值与Otsu相结合的方法。

4.公共地理数据辅助的云与高亮自然地表区分

  1. 摘要
    通过地理坐标将光学遥感影像与公开下载获取的公共地理数据相关联，直接判断光学遥感卫星影像上某位置是否为积雪冰原等高亮度地表，或者通过差异分析的方式定性识别影像中的不变场景，将它们否定为云，实现云与高亮度自然地表的区分。主要利用了云的时相特征。
  2. 其他的云雪区分方法
    分形维特征，“云-影”伴生关系，上下文场景分析，基于影像热红外信息（不适用于光学影像），基于多幅不同时间成像的影像（对输入数据的要求较高）
  3. 基于冰川分布图的冰雪场景识别
    在阈值分割得到的结果基础上，将高亮度目标点通过地理坐标，逐像素关联至冰川分布图中，从而认定其是否为冰雪。
  4. 基于Landsat-8影像的不变场景识别
    利用Landsat-8卫星的多波段数据，根据云的光谱特征，通过经验阈值做出6个判定，从而确定云雪
  5. Landsat-8无云控制影像制作
    数据的准备 -> 影像云检测 -> 无云区域合成
  6. 基于Wallis变换的差异分析
    得到Landsat-8无云控制影像后，对于带有地理坐标的正射影像，或者是配有PRC信息的传感器校正产品，便可通过地理关联的方式对两幅影像执行差异分析，进而区分不变场景和变化场景。其中，几何一致性大致可以满足，而色彩一致性需要先对影像进行亮度一致性处理，使得不含云的区域里，影像间的亮度区域尽可能小，因此使用Wallis线性变换法。最终再构造差异图，云仍然为亮，冰雪等高亮度地表则表现为灰色。

.多视角光学遥感卫星影像云检测方法

  1. 摘要
    以多视角成像的像对为研究对象，利用云层高度特征引起的投影视差，将多视角像对中的晴空场景看作不变的背景，将云层看作变化的、运动的目标，通过像对间的差异分析，辅之以亮度分析和形态学优化整合，实现了精确、高效的云与晴空场景的区分。主要利用了云的时相特征。
  2. 多视角成像原理及影像特点
    获取多视角影像的方式一般为多台相机同时成像。当成像夹角更大时，如前视和下视、前视和后视成像时，云与地表相对位置的变化将更明显。一些高大的山体也会出现投影视差，表现在影像上即高程起伏引起的像点位移，但高大山体的投影可以节主数字高程模型(DEM)进行修正。此外，成像夹角的存在还必然伴随着同一场景成像时间的差异。
  3. 基于投影视差分析的云检测方法
    　　1. 全色影像亮度一致性处理
    　　目的：使不含云的区域里，影像间的亮度差异尽可能地小。
    　　成因：在遥感影像地获取过程中，受到诸多因素的影响，加之前期辐射定标工作中可能存在的差异，同一区域范围内不同影像之间存在着亮度和色彩的差异。多幅影像间色彩一致性处理主要解决上述问题，又称匀色处理。
    　　思想：认为多幅待处理影像之间的色彩存在线性或非线性的映射关系，通过特定的数学模型模拟这种映射关系，将待处理影像的色彩校正至指定标准之上，或使其与选定的模板影像一致。
    　　方法：Wallis（线性校正法），直方图匹配法（非线性校正法）
    　　误差：多视角影像间未被完全消除的几何配准差；地形起伏和成像角度不同造成的影像局部光照不均匀。
    　　2. 多光谱影像的全色转换方法
    目的：找到合适的转换关系，将多光谱影像转为全色，使转换后影像与全色影像之间在晴空场景处的高度差异尽可能小。
    方法：波段平均法，HSL变换法，本文采用自定义的最小二乘法线性模型，通过影像金字塔和粗差探测的方式进行转换
    　　3. 像对差异分析与云区识别

6.云检测在遥感产品制作中的应用

  1. 摘要
    分析了云层对遥感影像处理过程中辐射校正、几何校正、合成影像制作、数字地表模型制作等环节造成不同影像，分别选择了对应的云检测策略，对光学遥感影像产品制作过程进行优化，更好地提取了海量影像中的有效、优质信息，提高了影像的利用率。
  2. 顾及云层干扰的影像辐射校正
    辐射校正指消除或改正遥感图像成像过程中附加在传感器输出辐射能量中的各种噪声的过程。
    1.含云影像条带噪声的去除
    　　光学遥感对地探测卫星获取影像的方式：推扫、摆扫
    　　条带噪声去除算法：直方图匹配法、矩匹配法
    2.单幅含云影像匀光处理
    　　Nask匀光法，又称模糊正像匀光法。它将内部亮度不均匀的遥感影像看作一幅受光均匀影像与一副亮度不均匀背景影像的叠加，通过低通滤波等方式近似模拟背景影像，再令其与原始影像做相减运算，实现原始影像的色彩一致性处理。
  3. 顾及云层干扰的影像几何纠正
    　　正射纠正：采集控制点 -> 求解定向参数 -> 基于定向参数和地表高程信息的影像重采样
    　　影像配准：采集控制点 -> 构建局部面元 -> 基于局部仿射变换的影像重采样
    　　特征点提取方法：Harris, SIFT。当云或积雪中的特征点较少时，采用的粗差探测算法，如基于最小二乘估计，基于随意抽样一致，基于残差分析的LMS估计法。
  4. 无云合成影像的制作

    　　多幅含云影像的匀色处理

《Densely connected deep random forest for hyperspectral imagery classification》

论文：https://www.tandfonline.com/doi/full/10.1080/01431161.2018.1547932

1.相关方法

  • extended morphological profile (EMP)
  • segmentation based method
  • spatial filtering
  • the sparse coding (to reduce the hyper-parameters)

2.模型优势

  • 参数少
  • 不需要太多的训练样本
  • 训练速度快，准确率高

3.DCDRF模型特点

训练阶段：

  • 每个像素点(pixel)会与其周围的4个邻居像素点(neighbouring pixel)一起被训练
  • 由于local smoothness of hyperspectral imagery，认定邻居像素和中心像素label值一样
  • 根据不同的相对位置，像素点会被投入不同的RF模型中训练
  • 其后根据不同的类别输出，像素点会被投入到不同的RF模型中训练
  • 最终输出对应不同分类的概率值
  • cross-validation - k
  • 因为使用了k折交叉验证，每个随机森林会为每个像素产生k个输出，所以文章将平均向量最为每个RF的输出

4.和gcForest的不同

#gcForest

  • the gcForest uses different random forests to increase the diversity
  • all the outputs of each random forest are concatenated to form the output of the current level
  • the output of the current layer is only used to augment the input of the next layer

#DCDRF

  • the current pixel and its neighbouring pixels are fed into the same kind of RF to improve diversity, performance and robustness
  • the average of each RF is used as the output
  • the output of the current layer will be used by all the subsequent layers to ensure maximum information flow between layers in the network

5.参数设置

  • The number of level: 6
  • The neighbouring size: 11 * 11
  • The tree number: 5

《基于多维多粒度级联森林的高原地区云雪分类》

论文：http://www.cnki.com.cn/Article/CJFDTotal-JSJY201808014.htm

1.改进的多粒度级联森林 (The improved gcForest)

  • 概要

    M-gcForest算法主要将训练分成了两个阶段：多维粒度扫描和级联森林阶段，收到神经网络中表征学习对原始特征进行逐层处理的启发，采用了深度网络的层叠结构，从级联森林前层输入数据，输出结果作为下层的输入。主要处理云雪区分问题。

  • 级联森林

    级联森林的每一层都是由决策树组成的森林组成的，每层都是“集成的集成”。
    当级联森林中的一层完成训练后，用此模型来对一个检验集进行预测，如果当前层的准确率比前一层的准确率高，那么继续构造级联森林的下一层，直至当前层对检验集的准确率较前一层的准确率不再提升，训练就会终止。
    因为级联森林的的单层之间的每个森林之间的训练是相互独立的，单个森林内部的各个决策树之间的训练也是相互
    独立的，这也就是gcForest 可以进行并行计算的原因。

  • 多维多粒度扫描

    将每个窗口滑到的数据都作为一个样本，从原始输入里面学习到输入特征向量，然后用两个随机森林输出的类向量作为新样本的表征。
    每个森林都会产生16个四维的类概率向量，最后将由不同粒度提取的类概率向量拼接起来，形成一个维度非常高的重新表示的特征向量。
    这个过程和深度神经网络有所区别，gcForest 对特征的提取是全程受监督的，而深度神经网络的特征提取过程是由高层的误差反向传播指导更新的。

利用M-gcForest 对多光谱数据进行训练和预测的效果比单光谱效果图好，这是因为M-gcForest 在多粒度扫
描阶段，其实是逐步取得窗口大小的像素块，反复地利用原始图像中的像素点之间的特征信息。

1.MODIS数据

MODIS仪器具有高光谱分辨率、高空间分辨率和高时间分辨率等特点，特别是具有36个光谱通道，光谱达到从可见光到热红外的全光覆盖。目前对MODIS数据的多光谱检测，一般根据不同的下垫面（一般为陆地、海洋、雪/冰、沙漠和高原）和不同种类的云（高、中、低云和直展云）采用不同的检测算法。当前成熟的MODIS数据的多光谱云检测算法主要分为基于可见光反射率、基于近红外反射率和基于热红外通道亮温及亮温差的算法。目前云检测算法往往通过多个单波段和波段间运算的阈值限制来检测云区，多个阈值的获取使得算法的主观因素增强，算法不稳定，只在某些特定情况下适用。

2.基于HSV色彩空间引入云检测领域

再MODIS波段1、6和26假彩色合成后的影像中，云再RGB色彩空间中为紫色和黄色，与其他地物色彩差别很大，容易区分。本文将MODIS波段1、6和26的假彩色合成(波段1作为R分量，波段6作为G分量，波段26作为B分量)进行HSV正交换 ，对获得的色调值(H)进行阈值限定，获得最后的云检测结果。
